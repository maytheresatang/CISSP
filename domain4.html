⚠️ TCP vs UDP — Exam Trap TCP = connection-oriented (3-way handshake: SYN → SYN-ACK → ACK). Reliable, ordered delivery. UDP = connectionless, no handshake. Fast but unreliable. Layer 4 connectionless = UDP (not ICMP — that's Layer 3).
IPsec Overview

IPsec operates at Layer 3 (Network). It provides authentication, integrity, and optionally confidentiality for IP packets.

AH vs ESP

🛡️ AH — Authentication Header

  • Provides: Authentication + Integrity
  • Does NOT encrypt payload
  • Protects IP header (including source/dest)
  • Protocol number: 51
  • Cannot traverse NAT (IP header changes)

🔐 ESP — Encapsulating Security Payload

  • Provides: Confidentiality + Auth + Integrity
  • Encrypts payload
  • Can traverse NAT (uses NAT-T)
  • Protocol number: 50
  • Most commonly used in practice
🚨 Critical Exam Rule: AH vs ESP Selection If the question mentions encryption or confidentiality → ESP. If only authentication/integrity → AH. If NAT is involved → always ESP (AH breaks with NAT). In practice, ESP is used for both because it includes authentication too.

Security Associations (SA) Count
📌 SA Count Formula Each SA is unidirectional. AH and ESP require separate SAs. For bidirectional communication using AH + ESP: 4 SAs (2 directions × 2 protocols). For ESP only (most common): 2 SAs.

🚨 Exam Trap: Know the encryption algorithm, not just the standard name WPA2 → CCMP → AES-128. WPA3 → GCMP → AES-128 (personal) or AES-256 (enterprise). TKIP is WPA — RC4 based, deprecated. The exam WILL ask for the algorithm name, not the standard name.
⚠️ SAE (Simultaneous Authentication of Equals) WPA3-Personal replaces PSK with SAE. Key improvement: forward secrecy — each session has a unique key, so capturing traffic today cannot be decrypted even if the passphrase is later compromised. SAE is resistant to offline dictionary attacks.

Memory Anchor
RADIUS = UDP + partial encrypt = Network Access · TACACS+ = TCP + full encrypt = Device Admin
When the exam describes securing a Cisco router CLI → TACACS+. When it describes 802.1X wireless auth → RADIUS.
Diameter Protocol

Diameter = successor to RADIUS. Used in 4G/LTE mobile networks and modern AAA infrastructure. Named because it is "twice the RADIUS" (same analogy, upgraded capability).

Kerberos
ComponentRole

AS (Authentication Service)Verifies identity, issues TGT.

Kerberos uses symmetric encryption and requires time synchronisation (default 5-minute skew tolerance). Port: 88.

IDS vs IPS

🔍 IDS — Intrusion Detection System

  • Detects and alerts — passive
  • Out of band — traffic copy analysed
  • Cannot block traffic
  • Lower risk of disrupting legitimate traffic

🚫 IPS — Intrusion Prevention System

  • Detects and blocks — active
  • Inline — sits in traffic path
  • Can drop/reset malicious sessions
  • Risk of false positives blocking legit traffic

⚠️ Exam Trap: Best DDoS mitigation The CISSP preferred answer for volumetric DDoS is CDN-based scrubbing (upstream filtering), not on-premise firewall. On-prem controls get overwhelmed before they can help.
